# Project 2 – Linux Fundamentals

## Objective

* Develop foundational Linux skills used by Security Operations Center (SOC) analysts by practicing file system navigation, file management, and basic investigation workflows within an Ubuntu virtual machine.

---

## Environment

* Ubuntu Desktop LTS
* Oracle VirtualBox
* Bash Terminal

---

## Skills Practiced

* Navigating the Linux file system
* Creating and managing directories
* Creating and managing files
* Viewing and editing text files
* Copying and moving files
* Removing files and directories
* Locating files using Linux commands
* Organizing an investigation workspace

---

## Linux Commands Used

| Command  | Purpose                                |
| -------- | -------------------------------------- |
| `pwd`    | Display the current working directory  |
| `ls`     | List files and directories             |
| `ls -la` | List all files, including hidden files |
| `cd`     | Change directories                     |
| `mkdir`  | Create a new directory                 |
| `touch`  | Create a new file                      |
| `nano`   | Create and edit text files             |
| `cat`    | Display the contents of a file         |
| `cp`     | Copy files                             |
| `mv`     | Move or rename files                   |
| `rm`     | Remove files                           |
| `rmdir`  | Remove an empty directory              |
| `find`   | Search for files by name               |

## Lab Activities

* Created an investigation workspace within Ubuntu.
* Practiced navigating the Linux file system using directory commands.
* Created folders to organize investigation evidence.
* Created and edited text files using Nano.

## Lessons Learned

* I learned how to confidently navigate the Linux file system using the command line.
* I practiced creating, viewing, copying, moving, and deleting files while maintaining an organized investigation workspace.
* I learned why preserving original evidence is important by creating copies before making changes.
* I also gained experience locating files quickly using the `find` command, an important skill during security investigations.

---

## SOC Analyst Takeaways

* Linux command-line skills are essential for Security Operations Center (SOC) analysts because many investigations are performed from the terminal.
* Organizing evidence and preserving original files helps maintain the integrity of an investigation.
* Commands such as `find`, `cat`, `cp`, and `mv` allow analysts to efficiently locate, review, and manage files during incident response.
